## 🎨 Customization Quick Tips

### Change Colors
Edit `src/App.css` lines 10-11:
```css
--primary-color: #6366f1;      /* Main color */
--secondary-color: #ec4899;    /* Accent color */
```

### Update Social Links
Search for these in the components and replace:
- `https://github.com` → Your GitHub profile
- `https://linkedin.com` → Your LinkedIn profile
- `https://twitter.com` → Your Twitter profile
- `https://instagram.com` → Your Instagram profile

### Add More Projects
Edit `src/components/Projects.jsx` - just add more items to the `projectsData` array!
